    // Enable the AI noise suppression feature
    agoraEngine.setParameters("{\"che.audio.enable.nsng\":true}");

2. **Set the <Vg k="RTEE_NS" /> mode**

    You can set one of the following modes for <Vg k="RTEE_NS" />:

    | Value | Mode  | Description |
    |:-----|:----|:------------|
    | 0 | Disabled | Disables noise suppression |
    | 1 | Deep-learning | Uses AI based deep-learning algorithm for noise reduction |
    | 2 | Hybrid | Uses both traditional and deep-learning noise reduction |

    To implement the desired mode, use the value from the table in the following code. Add this code to the `joinChannel` method after `agoraEngine.startPreview();`:

    ```java
    // Sets the mode of the AI noise reduction feature 
    agoraEngine.setParameters("{\"che.audio.ns.mode\":2}"); 
    ```
3. **Fully customize the <Vg k="RTEE_NS" /> settings**

        The default noise reduction effect is soft. To make the reduction effect more aggressive, add the following after `agoraEngine.setParameters("{\"che.audio.ns.mode\":2}");`:

    ```java
      // The limit of noise suppression gain mapping index. The default value is 80.
      agoraEngine.setParameters("{\"che.audio.nsng.lowerBound\":10}");
      // The limit of noise suppression gain mapping value. The default value is 50.
      agoraEngine.setParameters("{\"che.audio.nsng.lowerMask\":10}");
      // The limit of statistical noise suppression gain. The default value is 5.
      agoraEngine.setParameters("{\"che.audio.nsng.statisticalbound\":0}");
      // The gain limit of the whole noise suppression framework. The default value is 30
      agoraEngine.setParameters("{\"che.audio.nsng.finallowermask\":10}");
    ```
